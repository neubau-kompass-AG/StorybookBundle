# @sensiolabs/storybook-symfony-webpack

Webpack framework package for `sensiolabs/storybook-bundle`.

This package supports projects that need Storybook's Webpack 5 builder because they rely on Webpack-specific loaders, plugins, or Storybook customization. New projects can use either builder, with Vite generated by default.

Most users should install the Composer bundle and run:

```shell
bin/console storybook:init --builder=webpack
```

That command writes the Storybook config and points the project at this package. In Composer path installs it uses:

```json
"@sensiolabs/storybook-symfony-webpack": "file:vendor/sensiolabs/storybook-bundle/packages/webpack"
```

This package targets Webpack 5 because Storybook 10's Webpack builder is Webpack 5.
